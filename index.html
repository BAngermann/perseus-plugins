perseus-plugins

Perseus is a software framework for the statistical analysis of omics data. It has a plugin architecture which allows users to contribute their own data analysis activities. Here you find the code necessary to develop your own plugins.

Welcome to the Perseus plugin repository

Here you can find the interfaces that are required for the development of plugins for the Perseus software. Furthermore the source code for numerous standard plugins can be found here.

Perseus

Perseus is written in C# using the .NET Framework 4.5. Supported operating systems are

  • Windows Vista SP2 (x86 and x64)
  • Windows 7 SP1 (x86 and x64)
  • Windows Server 2008 R2 SP1 (x64)
  • Windows Server 2008 SP2 (x86 and x64)
  • Windows 8
  • Windows Server 2012

PerseusApi

This project contains the interfaces that the plugins according to which the plugins are programmed. IMatrixData represents the structure of the data that is passed from activity to activity. The five main types of activities correspond to the interfaces

  • IMatrixUpload
  • IMatrixProcessing
  • IMatrixAnalysis
  • IMatrixCombination
  • IMatrixExport
